# Cloud Budget Dashboard

A full-stack budgeting web application built with React and Supabase.

## Features
- User authentication (login/signup)
- Add and delete expenses
- Category-based spending tracking
- Interactive charts
- Persistent cloud database

## Tech Stack
- React (Frontend)
- Supabase (Auth + Database)
- PostgreSQL
- Recharts (Data Visualization)
- Vercel (Deployment)
